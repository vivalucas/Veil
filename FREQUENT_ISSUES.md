## Items are moved to the always-hidden section

This issue has been resolved in recent versions. If you are still experiencing this issue, please update to the latest version of Veil. If the problem persists, please [open an issue](https://github.com/vivalucas/Veil/issues) with details about which items are affected.

## Veil removed an item

Veil does not have the ability to move or remove items. It likely got placed in the always-hidden section by macOS. Option + click the Veil icon to show
the always-hidden section, then Command + drag the item into a different section.

## Veil does not remember the order of items

This issue has been resolved in recent versions. If you are still experiencing this issue, please update to the latest version of Veil. If the problem persists, please [open an issue](https://github.com/vivalucas/Veil/issues) with details about your setup.

## How do I solve the `Veil cannot arrange menu bar items in automatically hidden menu bars` error?

1. Open `System Settings` on your Mac
2. Go to `Control Center`
3. Select `Never` as shown in the image below
4. Update your `Menu Bar Items` in `Veil`
5. Return `Automatically hide and show the menu bar` to your preferred settings
